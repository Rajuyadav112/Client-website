# Click & Connect Media

Welcome to the official repository for **Click & Connect Media**, a Premium Digital Agency focusing on crafting digital legacies. We don't just manage social media; we build royal online empires, elevating brands and creators with strategy, elegance, and impact.

## 🌟 Overview
This repository contains the frontend source code for the agency's website. It features a responsive, high-end UI designed with modern web principles, smooth animations, customized cursors, and engaging glassmorphic layout elements.

## 🚀 Services We Provide
- **SEO (Search Engine Optimization)**
- **Social Media Marketing**
- **Product Photography**
- **Content Writing**
- **Web Development**

## 🏢 Industries We Dominate
- **Restaurant & Cafe**: Local SEO, Viral Reels, Influencer Events
- **Education & EdTech**: Lead Generation, Brand Authority, Success Stories
- **Luxury Real Estate**: Lead Gen Ads, Virtual Tours, Personal Branding
- **Healthcare**: Patient Search, Trust Building, Reviews Management
- **Manufacturing (B2B)**: LinkedIn B2B, Product Catalogs, Trade Shows

## 🛠️ Technologies Used
- HTML5
- CSS3 (Vanilla, Custom Properties, Glassmorphism)
- Vanilla JavaScript
- Font inclusion via Google Fonts (`Outfit`, `Syne`)
- Icons exclusively from [Ionicons](https://ionic.io/ionicons)

## 📁 File Structure
- `index.html`: The main landing page with hero, industries, services, and success stories.
- `about.html`: Details regarding the agency's legacy and mission.
- `services.html`: In-depth look at our digital marketing and media solutions.
- `blog.html` & `careers.html`: Secondary resource pages.
- `contact.html`: Form execution and contact details.
- `style.css`: All core styling, custom animations, variables, and responsive media queries.
- `script.js` / `cursor.js` / `fix_css.js`: Frontend logic, DOM manipulation, custom cursor handling and layout repairs.

## 🌐 How to view locally
Simply clone the repository and open `index.html` in your favorite web browser. No complex build pipelines or bundlers are required.

```bash
git clone https://github.com/Rajuyadav112/Client-website.git
cd Client-website
# Open index.html directly or serve using live server
```
